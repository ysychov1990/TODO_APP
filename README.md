# Projekt na przedmiot Projektowanie Interfejsów Użytkownika

W tym repozytorium znajdują się wszystkie pliki potrzebne do przedstawienia projektu. 

## Dokumentacja

Folder [docs/][docs/]: Pliki dokumentacji zarówno w formacie \*.md, jak i \*.pdf.


## Makiety Low-Fidelity 

Folder [frame0/][frame0/]: Wszystkie makiety Low-Fidelity znajdują się tutaj. Nazwa pliku sugeruje czy jest to design na urządzenia mobilne, czy komputery.


## Implementacja frontendowa

Plik `index.html` jest głownym plikiem implementacji frontendowej. Korzysta on również z zasobów znajdujacych się w folderach `css`, `img` oraz `JS`. 

### Stylowanie 
`index.html` korzysta z pliku `mainpage.css` jako głównego źródła styli. Plik ten tworzony jest jednak z plików znajdujących się w folderze `sass` z roszerzeniami `.scss`. Pliki te są kompilowane przy pomocy wtyczyki dostępnej w **Visual Studio Code** pod nazwą **Live Sass Compiler**
